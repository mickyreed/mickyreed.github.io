Sum Of A Gun

Retro first person shooter prototype

Prototype FPS game level made with C# in Unity, it is made in a retro 3D style with an aesthetic similar to Doom. This is a GreyBoxed Level design built to test out a State Machine to Control Enemy AI behaviour between idle, patrol, chase and attack. Two enemy variants have been created - Shooting and Melee Types, and a weapon System to define weapon class types and ammo. There is also a key-card System to unlock various doors between Areas.

Python Projects

Guess My Word

A Python Console Clone of the popular Wordle Game

This is a game that runs in the console, built with Python, it is a clone of the popular Wordle Game where the Player has to guess the daily 5 Letter Word in a limited number of guesses. This console version displays the correct guesses in color coded format depending on the correct location of the guessed letter. It also keeps a record of the letters guessed so far, and keeps a record of the high scores. Scores are based on time and number of turns to guess the answer.

Drone Commander

A Python/Django Tello Drone Controller with Custom CRUD to control DJTelloPy drone swarms

MyDroneControl is a Tello drone controller made for Diploma Advanced Programming. The app uses Django with a custom crud and template to control drone swarms using the DJITelloPy package. The app features the following:

  • Login and Register functionality for users
  • Create, update and Delete Drones and Swarms of Drones
  • Tello Drone Controller where you can view drones and what swarms they belong to
  • A dropdown menu to select a swarm to launch and a launch button
  • A Telemetry Flight Data feedback section which displays flight data related to the battery and flight time at the end of each launch.

Innovative Greenhouse Monitoring System

Harnessing IoT and Mixed Reality with Microsoft HoloLens

The project utilizes the power of Microsoft HoloLens, Azure, Unity (C#), and IoT technologies (Arduino & C++) to develop a cutting-edge solution for monitoring and maintaining optimal conditions in greenhouses and hydroponic farming systems. Designed for small-scale operations with the potential for adaptation to larger-scale applications, it offers a glimpse into the future of automated farming. This was a collaborative small team project providing an integrated solution for monitoring an existing hydroponic system, sending telemetry data to Azure IoT Hub and SQL database where it can be retrieved and viewed via a cloud based dashboard, and also be used to provide interactability with a digital twin model via use of Mixed reality on the Microsoft Hololens.
